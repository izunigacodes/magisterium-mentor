Términos de Servicio

Última actualización: 26/03/2025

Bienvenido a E-Magisterium. Al acceder y utilizar nuestra plataforma, aceptas cumplir estos términos y condiciones. Si no estás de acuerdo, por favor, no utilices el servicio.

1. Definiciones

  • Plataforma: El sistema académico disponible en www.e-magisterium.com.
  • Usuario: Toda persona que acceda y utilice el sistema (administradores, docentes, estudiantes, padres, etc.).
  • Servicio: Funcionalidades ofrecidas por la plataforma, incluyendo gestión académica, calificaciones, asistencia y comunicación escolar.

2. Uso del Servicio

  • Registro: Para acceder a ciertas funciones, debes registrarte y proporcionar información veraz y actualizada.
  • Responsabilidad del Usuario: Debes utilizar el sistema de manera lícita y respetuosa. No se permite el uso indebido o fraudulento.
  • Acceso y Disponibilidad: Hacemos nuestro mejor esfuerzo para mantener el servicio disponible, pero no garantizamos que sea ininterrumpido o libre de errores.

  • Está prohibido copiar, modificar o distribuir el software sin autorización expresa.

4. Protección de Datos

  • El uso de datos personales se rige por nuestra Política de Privacidad.
  • Implementamos medidas de seguridad para proteger la información de nuestros usuarios.

5. Suspensión y Terminación de Cuentas

  • Podemos suspender o cancelar cuentas que infrinjan estos términos, sin previo aviso.
